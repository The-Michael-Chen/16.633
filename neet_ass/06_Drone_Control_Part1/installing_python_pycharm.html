Installing Python and PyCharm/VSCode

1 Installing Python

To program the Tello drone, you’ll need to install Python on your computer. Although the Tello SDK 2.0 User Guide doesn’t specify a specifc version of Python, please download the latest stable version of Python 3 from the website www.python.org. Here, you’ll find detailed installation instructions for Windows, macOS, and other platforms.

During the installation process, click the option that says “Add Python 3.x to PATH” (see image below).

Note

  • Download Python 3 from the website www.python.org.

  • Click the option that says “Add Python 3.x to PATH.”

2 Choose a Python Editor

To edit and run Python programs on your computer, we suggest using a one of the popular editors such as PyCharm or Visual Studio Code. In previous years, students have successfully programmed their Tello with both of these IDEs.

2.1 Installing PyCharm

You can download PyCharm at https://www.jetbrains.com/pycharm/download/. Download the Community version, as it is free open source software (see image below).

When installing PyCharm, select the following options:

  • Add “bin” folder to the PATH

  • Create Associations .py

After installing PyCharm, you’ll need to restart your computer.

2.2 Installing VSCode

You can download Visual Studio Code (VSCode) at https://code.visualstudio.com/Download. It is free (see image below).

When installing VSCode, select the following options:

  • Register Code as an editor for supported file types

  • Add to PATH (requires shell restart)

After installing VSCode, you’ll need to restart your computer.
