Programming Using the Tello SDK 2.0

For this exercise you’ll learn how to write a program in Python using commands in the Tello SDK 2.0 User Guide . You’ll begin with existing code Tello_Example01.py, and once you have a grasp of how this code is structured, take a look at the User Guide to learn more about the other SDK commands. After looking through the User Guide, you’ll be in a good position to make modifications and create your own code.

1 Tello Example 01

First open PyCharm or VSCode and create a folder on your computer to place the program Tello_Example01.py. Next, copy the code below and place it inside of your folder.

#############################
# Tello EDU Sample Program 01 
#
# MIT NEET-AM Autonomous Machines
#
# 08/12/2021
#############################

import threading
import socket
import sys
import time

host = ''
port = 9000
locaddr = (host, port)

# Create a UDP socket
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

tello_address = ('192.168.10.1', 8889)

sock.bind(locaddr)

def recv():
    count = 0
    while True:
        try:
            data, server = sock.recvfrom(1518)
            print(data.decode(encoding="utf-8"))
        except Exception:
            print('\nExit . . .\n')
            break

recvThread = threading.Thread(target=recv)
recvThread.start()

try:
    msg = "command";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)

    msg = "takeoff";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msg = "down 20";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msg = "up 20";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msg = "flip l";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msg = "flip r";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msg = "cw 90";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)
    time.sleep(3)

    msgbat = "battery?";
    # Send data
    msgbat = msgbat.encode(encoding="utf-8")
    sent = sock.sendto(msgbat, tello_address)

    msg = "land";
    # Send data
    msg = msg.encode(encoding="utf-8")
    sent = sock.sendto(msg, tello_address)

    # close the socket
    sock.close()

except KeyboardInterrupt:
    print('\n . . .\n')
    sock.close()

After you’ve downloaded the progam Tello_Example01.py to your computer, follow the steps given below:

Step 1

Turn on your Tello and wait until it starts to flash yellow.

Step 2

On your computer, look under Network and find your Tello’s WiFi ID. Select your Tello. It may take a minute or two to connect, so be patient.

Step 3

Within PyCharm, look under the Run menu and select Run. Select the program Tello_Example01.py to run. If all works well, your Tello should execute the commands given in the code.

Step 3

rotate ccw 180 degrees

fly forward to (-100, 0, 100)

Step 5

rotate ccw 135 degrees

Step 6

fly forward to (0, -100, 100)

Step 7

rotate ccw 90 degrees

Step 8

fly forward to (100, 0, 100)

Step 9

rotate ccw 90 degrees

Step 10

fly forward to (0, 100, 100)

Step 11

rotate ccw 45 degrees

Step 12

fly forward to (-100, 100, 100)

Step 13

fly left to (-100, -100, 100)

Step 14

Step 15

rotate ccw 45 degrees

Step 17

Step 18

rotate cw 225 degrees

Step 19

land to (0, 0, 0)

Note

All of the Tello control commands for point-to-point flying are located on this page Tello SDK 2.0 User Guide.
